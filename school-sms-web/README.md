# School SMS Web (Vue 3 + Vite)

## Setup
1) Install deps:
   - `npm install`
2) (Optional) set API url:
   - create `.env` and set: `VITE_API_URL=http://localhost:8000/api`
3) Start dev server:
   - `npm run dev`

Login uses the Laravel API token.
